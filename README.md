# Personal Introduction Program

## 📌 Project Description
This is a simple Python program that takes user input like name, age, hobby, and city, and displays a friendly welcome message.

## 🛠️ Technologies Used
- Python

## 🚀 Features
- Takes user input using input()
- Stores data in variables
- Displays formatted output using f-strings

## ▶️ How to Run
1. Install Python
2. Open terminal / command prompt
3. Run the file:
   python personal_intro.py

## 📸 Output Example
What is your name? Alex  
How old are you? 21  
What is your favorite hobby? Coding  

🎉 Welcome Alex! 🎉  
You are 21 years old and love Coding.
